# IncodeVision Internship Tasks

This repository contains the tasks completed during my AI Internship at **IncodeVision**.  
Each task demonstrates practical applications of Python, Machine Learning, and NLP.

---

## 📂 Tasks Included

### Task 01 - Simple Chatbot
- A Python-based chatbot with predefined responses.
- Demonstrates basic input handling and conditional logic.
- Run: `python Task01_Chatbot.py`

### Task 02 - Spam Message Detection
- Detects spam vs. ham messages using Naive Bayes + TF-IDF.
- Demonstrates text preprocessing and classification.
- Run: `python Task02_SpamDetector.py`
- Dataset: SMS Spam Collection (spam.csv)

### Task 03 - Movie Recommendation System
- Suggests similar movies using cosine similarity on genres.
- Demonstrates recommendation systems and similarity measures.
- Run: `python Task03_MovieRecommender.py`

### Task 04 - AI Resume Screening System
- Matches resumes with job description using NLP similarity.
- Demonstrates text vectorization and semantic matching.
- Run: `python Task04_ResumeScreening.py`
